Principais Comandos Linux

Nesse artigo listarei alguns comandos para o terminal Linux que considero importantes e que uso com mais frequência. Básico do básico. Como não sou nenhum hacker, muito pelo contrário, uso o Ubuntu. Então, podem haver algumas variações com relação às distros que não são baseadas no Debian.

Comando Função
ctrl + alt + t Este na verdade é um atalho para abrir o terminal
pwd Mostra o diretório (pasta) em que você está
ls Lista os arquivos e subpastas dentro do diretório
ls -la Lista os arquivos de forma mais detalhada e também exibe ocultos
